As tags HTML (Hypertext Markup Language) são instruções que informam ao navegador como formatar o texto e definir a estrutura e o conteúdo de uma página web. As tags HTML são os blocos de construção fundamentais do HTML e são escritas entre colchetes angulares < >. Elas são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento.

Tags

O elemento i cria palavras em italico, ja a tag u coloca o undeline ou linha abaixo das palavras. Podemos utilizar tags para dizer se o texto será em negrito, por exemplo, utilizando a tag strong, outra tag que pode ajustar o tamanho, além dos Hs (h1, h2, h3, h4, h5 e h6) é a tag small
